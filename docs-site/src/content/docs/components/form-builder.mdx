## Pre-populating values

Call `SetValues(map[string]any)` on the built form to set initial field values programmatically — useful when editing an existing record.

## Relationship to Form

`FormBuilder` is a convenience layer over `Form`. All key bindings, submission, and cancellation behavior are the same — see the [Form](/components/form) docs for the full reference.
