## Key bindings

Table inherits core.Table's built-in navigation when `SetSelectable(true, false)` is active:

| Key | Action |
|-----|--------|
| `↑` / `↓` | Move row selection |
| `Home` / `End` | First / last row |
| `PgUp` / `PgDn` | Page up / down |
| `Enter` | Fire selection handler |
